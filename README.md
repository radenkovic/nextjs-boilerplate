# NextJS Boilerplate

Includes:
- NextJS / React
- Eslint (Airbnb+Tailwind+Prettier+Cypress)
- Prettier
- Tailwind
- Authentication via cookies (stateless)
- Cypress (e2e tests)
- API Client (with autoloading tokens)
- React-form for forms
- React-query for query-ing (with caching!)

Routes:
- `/` non protected
- `/about` sample of protected route
- `/login` login route
